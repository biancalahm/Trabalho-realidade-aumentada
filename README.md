# Trabalho-realidade-aumentada
Este repositório refere-se ao trabalho de realidade aumentada 
APK unity:  Insira o usuário @teste e um id valido no campo senha (1 ao 14)
Os marcadores presentes na solução para cada tela do museu estão no arquivo marcadores.pdf

Pre requisitos instalar aplicação:
Unity 20019.1.14
Este projeto esta conectado a um servidor remoto \\

1 - Instalar pacote da pasta unity guia_museu_bianca_ natalia.unpackage no unity \\
2- A rota do servidor está configurada no arquivo Assets -> App -> Resources-> linkServer.cs \\
3- Ajuste o game resolution para para Portrait caso esteja em outro formato \\
4- Adicione as cenas ao build  do projeto unity \\
5- Execute a cena acessar atração com o usuario @teste as senhas variam de 1 á 13 \\

Os marcadores presentes na solução para cada tela do museu estão no arquivo marcadores.pdf\\
